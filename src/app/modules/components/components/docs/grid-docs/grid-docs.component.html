Examples

From the stack to the horizontal arrangement.

The default grid system is 24. Allows you to define a system with a different value.

Layout

Child elements depending on the value of the start, center, end, space-between, space-around, space-evenly, which are defined in its parent node layout mode.

From the stack to the horizontal arrangement.

You can create a basic grid system by using a single set of ngx-row and ngx-col grid assembly, all of the columns (ngx-col) must be placed in ngx-row.

Alignment

Child elements vertically aligned.
