Button

To trigger an operation.

When To Use

  • A button means an operation (or a series of operations). Clicking a button will trigger corresponding business logic.

Examples

Basic
Basic usage of button.
Three sizes
[ngxSize] represents three types of size.
Allows color change
You can change the color by using [ngxColor].
Four types of fill mode
[buttonFillMode] represents four types of fill mode.
Four types of border radius
[buttonRounded] represents four types of border radius.
