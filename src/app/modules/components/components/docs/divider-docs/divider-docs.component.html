Divider

A divider line separates different content.

When To Use

  • Divide sections of article.
  • Divide inline text and links such as the operation column of table.

Examples

Horizontal

Divider default type is horizontal. Support inner text inside Divider.

Orientation of title

Set orientation of divider to left or right.
Allows color change.

You can change the color by using (ngxColor).
Vertical

Use ngxType="vertical" make it vertical.
