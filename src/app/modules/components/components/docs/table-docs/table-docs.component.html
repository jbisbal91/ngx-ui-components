Table

A table displays rows of data.

When To Use

  • To display a collection of structured data.
  • To sort, search, paginate and filter data.

Examples

Basic

Basic usage.
Multiple sorter

ngxSort directive to set the priority of sort columns. (changeSorting) to customize the comparison function.

Resize columns

ngxResize directive to resize column sizes.
All features

In this case, all possible variants of the table are represented with the incorporation of components in its cells.
